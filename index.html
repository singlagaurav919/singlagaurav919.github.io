The Buggy Project

Tech Stack - C language

IDE - Arduino Software

I along with my team built this for the project work of Engineering Design course during my bachelors.

We made an obstacle avoiding robot which can also follow a given path. We implemented this on a buggy with the Arduino Uno board and programmed it in C language using Arduino Software. We used an IR sensor to make the buggy follow a pre-described path. For obstacle detection, we used an ultrasonic sensor that reflects radiation from an object and notifies the buggy for redirection.
